# iMicroSeq Data Collection Template

This folder relates to the implementation of the **iMicroSeq contextual data specification**, which has been developed based on findings from the needs assessment. The specification defines the core data fields and controlled terms required to support interoperable data capture for water-based microbial genomics.

The full set of fields, definitions, and terms that comprise the specification are described in the **Reference Guide**.

## Accessing Templates
This specification is designed to be used with the DataHarmonizer application. To use the iMicroSeq template to curate your data, please go to the most recent pathogen genomics package.

*At present, the iMicroSeq templates are not yet available in the Pathogen Genomics Package (PGP) repository. Integration is in progress and expected in the coming weeks. Once available, users will be able to interact with the templae within the DataHarmonizer interface.*

Alternatively, users can create their own templates using the fields and controlled vocabularies defined in the Reference Guide.
